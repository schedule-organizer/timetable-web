# Epic 4 - Guided Onboarding & Setup Templates

**Goal:** Shorten time to value for new institutions with pre-structured setup paths and reusable defaults.

**Primary personas:** Timetabler, Principal

**Business value:**
- directly supports the MRD goal of reducing time to first timetable
- differentiates SchediFlow from rigid or expert-only competitors
- improves free trial activation and conversion

**Key capabilities:**
- institution-type template selection
- guided first-run setup path
- prefilled defaults for common school models

**Frontend alignment:** `TMPL-05`

**Success signal:** first-time users can start from a template instead of configuring every concept from scratch.
