# Daily Operations Mockups

This family covers the day-of timetable workflows that sit between the timetable itself and the school's live operations.

## Pages

1. `index.html` - workflow hub
2. `cover-management.html` - mark absences and assign cover
3. `delegation-request.html` - teacher request flow
4. `delegation-queue.html` - moderator approval queue
5. `temporary-schedules.html` - special-week timetable setup

## Notes

- Moderator pages use the shared session header with moderator actions.
- The delegation request page uses the shared session header with teacher actions.
- This family reuses the root design tokens from `../styles.css`.
