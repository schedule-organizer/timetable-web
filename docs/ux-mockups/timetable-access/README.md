# Timetable Access Mockups

Shared timetable access workflows for published schedules and personal timetable views.

Pages:
- `index.html` - hub for the timetable access family
- `publish-timetable.html` - publish confirmation and status
- `master-timetable.html` - full timetable workspace
- `teacher-timetable.html` - personal teacher timetable
- `student-timetable.html` - personal student timetable
- `parent-timetable.html` - read-only parent timetable

These pages belong in the shared mockup area because they are not admin-only or moderator-only flows.
